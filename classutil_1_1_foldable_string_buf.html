Detailed Description

入れ子深さに応じた段下げ表示を行うよう拡張された文字列バッファのクラス.

段下げ用に、1組のメソッド {push()}と {pop()}をもつ. 文字出力用に、メソッド {nl()}、 {space()}、 {str(String s)} をもち、これらは不要な空白や改行を抑制し、現在の入れ子深さに応じた段下げ表示（indentation）を行う. 入れ子データの清書プログラムNodeFormattableに使用する.
