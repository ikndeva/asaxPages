Detailed Description

カッコによる入れ子文字列（ballanced parentheses strings）の清書プログラム.

///

コマンドラインまたは、オブジェクトとして使用できる. コマンドライン実行の利用法は次の通り.

usage: Par [-d ARG] [-e] [-p ARG] [-s ARG] [-t ARG]
-d,--delimitor char ARG specify a character for the list
delimitor. (default ';')
-e,--end-block folding switch folding of the close/end
parenthesis (default true)
-p,--parenthesis-delimitor ARG A two-letter length string such as
'{}' to specify a pair of
block-delimitors. The default is '()'.
-s,--stem string ARG specify a string for a stem for
indent. (default '|')
-t,--input text ARG specify an input text as a string
(default disable)</pre></li></ul></li></ul>
