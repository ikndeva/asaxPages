Detailed Description

インタフェースResettableをもつ型Tのオブジェクトのリサイクル可能なオブジェクトプール 次のように使用する．

RecycleManager<StringBuf> man =
new RecycleManager<>(
()->new StringBuf(), //generator
(StringBuf obj)->obj.clear() //resetter
);
StringBuf buf = man.get();
String str = buf.clear().add(...)...add(...).toString();
man.recycle(buf);
