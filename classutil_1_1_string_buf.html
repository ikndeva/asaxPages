Detailed Description

文字列バッファのクラス.

再利用可能な文字列バッファのクラス. 再初期化用のメソッド{clear()}をもつ以外は、java.lang.StringBufferと同等である. 基本的な使用法は，次のとおりである:

  • 最初にメソッド clear() で初期化する．
  • 続けて繰り返しメソッド add(String s)を用いて文字列や値を追加する.
  • 最後にメソッド toString() でjava文字列を生成して返す． 実装は，java.lang.StringBuilderを用いており， java.lang.StringBufferより軽量である．
