Detailed Description

イベントのメモリ管理モジュールのクラス.

オブジェクトの取得時に、Event getNewEvent()をnew Event()の代わりに呼び出し、使用済みのイベントeに対してvoid recycleEvent(Event e)を呼び出すことでメモリ管理を行う.
