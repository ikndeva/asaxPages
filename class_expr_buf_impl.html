Detailed Description

template<class T>
class ExprBufImpl< T >

式用の入れ子段下げした清書出力を行うためのバッファ。

利用では，CRTPパターンによる再帰的クラス定義を用いたExprBufクラスを使用すること．

対象のブロック（または要素）のレベルに応じて、ブロックの開・閉に位置で出力の段下げ（折り曲げ）を行なって清書出力する。
例として、次の出力コードに対して、

cout
<< endl << "@exp input: lv=" << lv
<< buf
.open("proc Rec(node *v) { ")
.open("if (v->leaf()) { ")
.leaf("return; ")
.close("} ")
.open("for (child in v.children()) { ")
.leaf("Rec(child); ")
.close("} ")
.close("} ")
.str();

次の清書出力が得られる。

proc Rec(node *v) {
if (v->leaf()) {
return;
}
for (child in v.children()) {
Rec(child);
}
}


インライン（inline）モードでは、ブロック折り曲げ（folding）をせずに、次のような清書が可能である。インラインモードは、手続き inl(true) を用いてブロック単位で設定できる。

proc Rec(node *v) { if (v->leaf()) { return; } for (child in v.children()) { Rec(child); } }


ノーエンド（no_end）モードでは、ブロックの終端を折り曲げずに、次のような清書が可能である。 ノーエンドモードは、手続き set_no_end(true) を用いてブロック単位で設定できる。

##### EXP2: no_end blocks by level #####
@exp input: lv=0
proc Rec(node *v) {
if (v->leaf()) {
return; }
for (child in v.children()) {
Rec(child); } }
