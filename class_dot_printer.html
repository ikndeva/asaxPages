virtual std::string suf_to_dot ( NT *  v,
StylePen *  pen 
)
inlineoverrideprotectedvirtual

dot形式で接尾辞リンクを記述する文字列を返す。

  • 実装メモ。Dotツールがきれいに配置するには、入力グラフがDAGであることを要する。
  • そこで、接尾辞リンクは逆向きの(dest -> src)順で描画し、矢印の表示を属性"dir = back "で逆向きに表示することでグラフをacyclicにする。
