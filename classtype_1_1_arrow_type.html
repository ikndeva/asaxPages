Detailed Description

基本的な関数型のクラス

関数型t_0 x ... x t_{n-1} -> tを表す型項Tを実装する． ここに，ノードの返値（自身）の型tと，入力値（子）の型t_0 x ... x t_{n-1}は，相異なる同値類に属する．
