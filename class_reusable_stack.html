Detailed Description

template<class S>
class ReusableStack< S >

型変数 S のメモリオブジェクトをもつ再利用可能なメモリオブジェクトのスタック。

  • メモリスタックに格納された各要素は自動的に生成され、 push操作による伸長時の再初期化（リセット）され、 pop操作による縮小時に隠蔽されて、メモリスタックの寿命の間は再利用される。
  • 実装メモ：内部のスタックの各エントリは、メモリオブジェクトへのポインタを保持する。
