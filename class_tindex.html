Detailed Description

template<class NT>
class Tindex< NT >

ノード型NTをもつグラフを用いたテキスト索引のクラス。

  • 原子辺(atomic edge)をもつグラフ索引のための描画ルーチンを提供する。
  • ここに、原子辺とは、文字列長さ1のラベル、すなわち、文字ラベルをもつ木辺（goto link）と接尾辞辺（suffix link）をいう。

NT *  newnode 
)
inline

ノード node からの接尾辞リンクが指すノード newnode に対して、そののslinkのラベルの先頭位置を設定する。

ノード newnode のslinkラベルの先頭位置 newnode->sbeg は、元のノードのslinkラベルに対して、その先頭位置を、node と newnode のLPT木の深さの差だけ、後ろへ移動させて得られる。
