Detailed Description

template<class NT>
class App< NT >

ノード型NTをもつテキスト索引のドライバークラス。

頂点ベースのグラフ索引用のクラスで、実行に必要な各種のメソッドを提供する。 このクラスは、コマンド入力を与えることで、次のオプションと入力を受け付ける。 

usage: ./main.exe OPTIONS
EXAMPLES
% echo 'abc' ./main.exe
OPTIONS
-h   print this help
-i   input file name
-n   non-verbose mode. set trace level = 0
-v   verbose mode (default)
-t level     set trace level (default level = 1)
EXPERIMENTAL
-g   draw the graph of an index into file

使用例を示す。 

#include "app.h"
#include "strie.h"
int main( int argc, char* argv[] ){
  App<MapPlainNode> driver;
  Sindex<MapPlainNode> *G = new STrie(); 
  driver.run(G, new DotPrinterNode(), argc, argv);
  return 0;
}; //main
