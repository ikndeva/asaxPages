static int bitWidth ( int  val)
static

与えられた整数値{val}の最小ビット幅を返す.

ここに、最小ビット幅とは、バイナリ表現のときに、位置0ビットから最上位のビット位置 {MSB}の幅{width}である. ここに、区間{width := |[0..MSB]| = MSB + 1}であり、

  • 整数値{@val}が0のときは0を返す.
