Detailed Description

ログ出力用のトレースレベルを扱うクラス。

簡易型のログ出力用ユティリティ。トレースレベルは、列挙型Levelの要素を用いる。サンプルプログラムを次に示す.
<quote>

void main(String[] args) {
    TraceLevel log = new TraceLevel(); 
    //コマンドラインのオプションを解析する; 
    if (コマンドラインのTRACEオプションが真なら) log.setLevel(TraceLevel.TRACE)

    if (log.LEVEL(TraceLevel.NONE)) ログメッセージを出力する;  //出力されない
    if (log.LEVEL(TraceLevel.VERBOSE)) ログメッセージを出力する; //出力されない
    if (log.LEVEL(TraceLevel.TRACE)) ログメッセージを出力する;   //出力される
    if (log.LEVEL(TraceLevel.DEBUG)) ログメッセージを出力する;   //出力される
}

</quote>
