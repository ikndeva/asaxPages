Detailed Description

整数型変数と文字列変数のカウンタをもつ計数器オブジェクトのクラス。

現在，次の型のカウンタを実装している：

IncCounter countBoolean = null; //論理型変数のカウンタ
IncCounter countInteger = null; //整数型変数のカウンタ
IncCounter countDouble = null; //整数型変数のカウンタ
IncCounter countString = null; //文字列変数のカウンタ
