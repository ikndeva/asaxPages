Detailed Description

基本型を表す半順序をもつ列挙型のクラス。

nullは型がないことを表す．

値型xt0と値型xt1の最小汎化（最小上界）、すなわち、xt0とxt1のいずれよりも大きい（両者を包摂する）ような値型の中で最も小さい（具体的な）ものを返す。 最大元Top=Undefと最小元Bottomをもち、{{Boolean}, {String}, {Number, Double, Integer}}の場合で、各ブロックないでだけ全順序で比較可能な特殊な場合のみ正しい。

Note
この実装は、一般の束では働かない。
