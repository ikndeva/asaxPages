Detailed Description

式がもつ値の型のクラス．Kotlin言語の型システムを参考にしている． 次の型をもつ．

- Boolean, Integer, Double, String: 基底型(concrete type)
- Any: 最汎型．旧:Bottom型．どの方にもマッチする．
- Unit: 単一値型．一意な値unitの型．文の返り値の型．旧: Unit型．
- Nothing: 最小型. 矛盾の型．例外の型．
- OStream: 出力ストリームの型．Stream(String) = String型で代用する．

値型のパッケージ
Definition: ArrowType.java:4

型階層の半順序>=は，最汎型Anyを最大元とし，矛盾型Nothingを最小限とする． 型推論は，最終的に全ての式が具体型（concrete type）を持つことを目標とし，解決できなければ失敗する． 型階層は，次の通り：

MostGeneralType := Any
Any > NumberOrString, Boolean, String, Unit
NumberOrString > Number, String
Number > Integer, Double
ConcreteType := {Integer, Double, Boolean, String, Unit} > Nothing
