Detailed Description

ノードの表示スタイルに関するプロパティのクラス.

各プロパティの意味は次の文法が示すとおりである：

OBJ := FUNC LPAR CHILD_1 OPER_1 ... OPER_K-1 CHILD_K RPAR
FUNC := [funcLPAD] funcSTR [funcRPAD]
LPAR := parenLPAR [parenLPAD] [parenLFold]
RPAR := [parenRFold] [parenRPAD] parenRPAR
CHILD_i := [childLPAD] childSTR [childRPAD]
OPER_i := [operLPAD] operSTR [operRPAD]
