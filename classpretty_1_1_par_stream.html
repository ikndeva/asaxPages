Detailed Description

カッコによる入れ子文字列（ballanced parentheses strings）の清書プログラム.

コマンドラインまたは、オブジェクトとして使用できる.

  • コマンドヘルプ: コマンドライン実行の利用法は次の通り. 
    • usage: ParStream [-d ARG] [-e] [-p ARG] [-s ARG] [-t ARG]
 -d,--delimitor char ARG          specify a character for the list
                                    delimitor. (default ';')
 -e,--end-block folding             switch folding of the close/end
                                    parenthesis (default true)
 -p,--parenthesis-delimitor ARG   A two-letter length string such as
                                    '{}' to specify a pair of
                                    block-delimitors. The default is '()'.
 -s,--stem string ARG             specify a string for a stem for
                                    indent. (default '|')
 -t,--input text ARG              specify an input text as a string
                                    (default disable)
  • 実行例: コマンドラインの実行例. 
    • $cat input.txt | java java -cp "." ParStream -e 
$java -cp "." ParStream -t '(A (B C) (D E F))' 
(A
| (B C)
| (D E F)
)
  • 実行例: Javaプログラムからの利用例. 
    • String input_str = null; 
input_string をセットする. 
ParStream pp = new ParStream();
pp.readOptionAndArgs(args);

if (pp.cmd.hasOption("t")) { //input from a given string
    System.out.println(pp.fold(input_str));
}
else { //input from an input stream 
    pp.run(System.in, System.out);
}
