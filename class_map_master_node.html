Detailed Description

MapPlainNodeに対応するマスターノードのクラス。

  • マスターノードは、任意の文字 ch に対して、メソッドget(ch)は一意な根を返す。
  • その唯一の子をprivate 変数 Node *the_single_child に保持する。

与えられたノード vを唯一の子ノードとする。

  • 文字 any_char は索引の描画の際の辺ラベルに用いる。
  • この際に、添え字にどのような文字 ch を与えられても関係なく、与えられたノードを特別な唯一の子 the_single_child として保持する。
