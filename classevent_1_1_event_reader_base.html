Detailed Description

実際の走査器は、本クラスを継承して、作成する.

Member Data Documentation

◆ EXACT_TEXT_MODE

boolean EXACT_TEXT_MODE = false

テキストイベント検出のふるまいを定めるブール値フラグ.

デフォールトはEXACT_TEXT_MODE = falseであり、テキスト要素が空文字列のときはイベントを検出しない. EXACT_TEXT_MODE = trueのときは、テキスト要素が空文字列のときもイベントを検出する.
