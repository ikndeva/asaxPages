Detailed Description

template<class NT>
class VisitableMaster< NT >

NTに対応するマスターノードのクラス。

  • マスターノードは、任意の文字 ch に対して、メソッドget(ch)は一意な根を返す。
  • その唯一の子をprivate 変数 Node *the_single_child に保持する。
