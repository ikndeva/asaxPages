Detailed Description

EventReadableインタフェースを実現するストリーム指向のプル型XMLパーザのクラス.

入力のXMLテキストをメモリに保持することなく，先頭から末尾へ一方向に読みながら，STAGと，ETAG，TEXTなどのイベントを検出する．文書の入れ子レベルに合わせて，読み書き可能なキー・バリューストア（変数環境）であるEnvを管理する．このEnvを応じて，post-order順の巡回において，現時点で仮想的に訪問している文脈頂点において，その頂点のラベル（タグ名と属性名）や値（テキスト値と属性値）などの情報を取り出すことができる．

Javaプログラムからの利用例.

import java.io.*;
import event.*;
EventReadable eis =
new AsaxEventReader(
new BufferedReader(
new InputStreamReader(System.in, "UTF8")));
//メインループ：入力ストリームを走査する
for (Event ev: eis) {
;//イベント ev を用いて何かする
}
eis.close();
