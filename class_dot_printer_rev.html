int  top_depth_ 
)
inlineoverrideprotectedvirtual

ノードに逆辺のグラフでの深さを設定する。

  • グラフを巡回して、接尾辞リンクに関する深さを、各頂点に設定する。
  • ここに、各頂点の深さは根からその頂点までの接尾辞リンクの長さ（枝数）で定義する。
  • 根の深さは0である。
