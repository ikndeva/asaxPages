Detailed Description

値を保持するオブジェクトのインタフェース

返り値型 T をもつオブジェクトとして再帰型 S を定義する。 使用例として、クラス定義では、自分自身の型 S と返り値型 T を与えて、次のように再帰的に使う：

class IntegerHolder implements Container<Integer,IntegerHolder> {...}
Note
実装のメモとして、このインタフェース定義は、自分自身を返すメソッドをテンプレート機構で定義するために、再帰テンプレートパターン（CRTP, Curiously Recurring Generic PatternAdd Star)を用いている。 インタフェース Container の実際のインタフェースは、次のように定義されている。
public interface Container<T, S extends Container<T,S>> {
public S reset();
public T get();
public String toString();
}
この定義で、自分自身の型がS extends Container<T,S>と再帰的に定義されているのがポイントである。 これにより、カスケードスタイルの呼び出しをもつメソッド public S reset() を、自分自身の型 S を返り値型としてもち、自分自身を返すメソッドとして定義することができる。
