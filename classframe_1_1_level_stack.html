Detailed Description

レベル付きスタックのクラス。

レベル付きスタックは、次の演算をもつ任意のStackableインタフェースのスタックVに、レベル区切りを付けたスタックである。

interface Stackable {
void push_(); // size()を1増加させる。
void pop_(); // size()を1減少させる。
void clear(); // size()=0にする。
int size(); // 非負整数であるサイズを返す。
}
