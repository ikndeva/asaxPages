Detailed Description

YAML書式での出力を行うクラス.

メソッドは，返り値として自分自身の参照を返し，ストリームスタイル（後置記法）で続けて記述する． オブジェクトの区切り記号はYamlBufオブジェクトが適宜挿入する． オブジェクト文字列を構築は次の手順で行う．

  • メソッドopenObj()でオブジェクトの記述を開始する．
  • メソッドadd(属性名，属性値)で属性値対をくりかえし追加する．
  • メソッドcloseObj()でオブジェクトの記述を終了する．
  • 最後にメソッドtoString()で文字列に変換して，使用する．

オブジェクト文字列を構築する際は，メソッドopenObj()とcloseObj()で記述を開始および終了する． 配列文字列を構築する際は，メソッドopenArray()とcloseArray()で記述を開始および終了する． オブジェクト記述と配列記述は，自由に混在して，任意の段数の入れ子にできる． 実装には，複数のFoldableStringBufを用いている．

例：

YamlBuf yaml = new YamlBuf();
  System.out.println(yaml.openArray()
         .add(1).add(2).closeArray().toString()); 
  System.out.println(
         yaml.clear()
         .openObj()
         .add("numEventRead", this.numEventRead)
         .add("text", "'"+text+"'")
         .add("fid", rule.state())
         .closeObj()
         .toString()
  );
