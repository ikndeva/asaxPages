Detailed Description

YAML書式の文字列を構築するための一連のメソッドを提供する． メソッドは，カスケードスタイルのインタフェースをもち，YamlBufオブジェクトの後ろに後置のドット'.

'を用いた記法で，複数のメソッド呼び出しを連続して書く．

  • オブジェクトの構築は，メソッドopenObj()とcloseObj()で開始と終了する．
  • 配列の構築は，メソッドopenArray()とcloseArray()で記述を開始および終了する．
  • オブジェクトと配列の構築を自由に混在し，任意の段数の入れ子にできる．インデントと区切りは，入子に応じて，自動的に挿入される．
  • オブジェクト文字列を構築は次の手順で行う．
    • メソッドopenObj()で，バッファを開き，オブジェクトの記述を開始する．
    • メソッドadd(属性名，属性値)で属性値対をくりかえし追加する．
    • メソッドcloseObj()でオブジェクトの記述を終了する．
    • 最後にメソッドtoString()でバッファを閉じて，内容の文字列を返す．
      Note
      複数のFoldableStringBufKtを用いて実装されている．
      例：

      System.out.println(yaml.openArray()
      .add(1).add(2).closeArray().toString());

      yaml.clear()

      .closeObj()
