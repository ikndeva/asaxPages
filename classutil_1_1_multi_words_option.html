Detailed Description

アパッチCLIのオプションパーザーのヘルパークラス．

オプションパーザー（apache.commons.cli.Options）を利用する際に，空白を含む文字列をオプション引数として渡せるように設定したオブジェクトを返すファクトリクラスである． 実装では，オプション生成時に，フラグcli.Option.UNLIMITED_VALUESを設定し，引数読み出し時に，空白で区切られた複数値を空白をはさんで連結した文字列を返すことで，オプションに可変長文字列を渡せるようにする．
使用法：アパッチCLIでは，通常はコンストラクタを用いて，クラスapache.commons.cli.Optionオブジェクトを生成する：

org.apache.commons.cli.Option opt = new Option("e", "query", HAS_ARG, "Setting a query string");
opts.addOption(opt)

その代わりに，ファクトリ手続きMultiWordsOption.createOptionを使って，Optionオブジェクトを生成する:

org.apache.commons.cli.Option opt = MultiWordsOption.createOption("e", "query", HAS_ARG, "Setting a query string");
opts.addOption(opt);

オプション引数の読み出しは，静的手続き MultiWordsOption.getOptionValue を用いて，空白区切の複数個の文字列を，間に空白をはさんで連結して得られる単一文字列を返す．
使用例は次の通り．

import org.apache.commons.cli.*; //Apache CLI library
public static void main(String[] args) {
apache.commons.cli.Options opts = new apache.commons.cli.Options();
opts.addOption(MultiWordsOption.createOption("e", "query", HAS_ARG, "a query string"));
CommandLine cmd = this.optParser.parse(opts, args);
if (cmd.hasOption("e"))
String queryString = MultiWordsOption.getOptionValue(cmd, "e");
}
